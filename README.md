# Simple_REST_API
Simple REST API that allows for a handful of HTTP requests.


# Clone the repository
```bash
git clone git@github.com:HenryNgai/Simple_REST_API.git
```

# Run the application
```bash
cd Simple_REST_API
go run ./cmd/api/main.go
```
